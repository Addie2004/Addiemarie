## 🚀 Getting Started

### Prerequisites
- Git installed on your computer
- A GitHub account

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/Addie2004/Addiemarie.git
   cd Addiemarie
   ```

2. Start making changes and learning!

## 📝 Commit Message Guidelines

I'm practicing writing clear commit messages:
- Start with a verb: "Add", "Fix", "Update", "Remove"
- Be specific about what changed
- Example: "Add README with project structure"
